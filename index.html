Exploratory Data Analysis (EDA)

After scrubing the data set, EDA is performed to help us to have better understanding on our dataset. Some example will be:

-> Information of dataset - what is the data type for each column?

-> Descriptive statistic - how the data in each column distributed?

-> Plotting - Bar chart/line chart/scatter plot etc. for visualization

-> Heat map - observe the correlation in between target variables and other dependent variables

With this, we can proceed to classification or regression model for further analysis.

Classification Model

Telco Churn Analysis Go to Notebook

Nowadays, every types of businesses are having fierce competition with various of competitors. Same happens to telco company. For example, in Malaysia, we have Digi, Maxis, Celcom etc. They are having great competition in terms of packages, services and technology. From business side, telco company need to know what's the existing behaviour while targeting to attract more new customers. Hence, telco churn analysis is to analyze and help telco company to

-> understand what is the current strength

-> analyze whether their customers will churn or not and what's the reason contributing to it
